Administration Tips

  • Wipe Data is non-destructive to credentials. Use it if a user wants to "start fresh" or if they've hit storage limits inappropriately.
  • Storage Usage includes recipe PDFs, history, inventory, and library JSON files.
  • User Deletion is final. It removes both the database entry and the entire state directory.

Server Security

Registration is currently locked with a Secret Key. You can modify this key or the admin password anytime in your server's .env file. Changes take effect on server restart.
